# finance-helper
Данный репозиторий содержит дипломную работу на тему «Разработка мобильного приложения для управления личными финансами на платформе Android». В рамках работы реализовано приложение-помощник для учета доходов и расходов, планирования финансовых целей, управления бюджетом и формирования отчетов. Приложение разработано в среде Android Studio с использованием языка программирования Kotlin.

Автор работы: Махмутов Данил, ПИ-4, Дизайнер

This repository contains a thesis on the development of a mobile application for personal finance management on the Android platform. The work implements a financial assistant application for tracking income and expenses, planning financial goals, managing budgets, and generating reports. The application was developed in Android Studio using the Kotlin programming language.

Author: Danil Makhmutov, PI-4, Designer
